## main layer

```
┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬───────────────┐
│noop │ f1  │ f2  │ f3  │ f4  │ f5  │ f6  │ f7  │ f8  │ f9  │ f10 │ f11 │ f12 │  noop         │
├─────┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬───────────┤
│   Tab   │  q  │  w  │  e  │  r  │  t  │  y  │  u  │  i  │  o  │  p  │noop │noop │  noop     │
├─────────┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴─┬───┴───────────┤

├───────────┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴───┬─┴────┬┴───────────────┤
│    noop       │  z  │  x  │  c  │  v  │  b  │  n  │  m  │  ,  │  .  │  /   │    noop        │

│ noop │ Fn │ noop  │noop │                  Space/sym                     │noop │ Opt │ noop │
└──────┴────┴───────┴─────┴────────────────────────────────────────────────┴─────┴─────┴──────┘
```

| key | description |
| --- | ----------- |
| noop | No operation |
| esc | Escape |
| M | Meta |
| A | Alt |
| S | Shift |
| C | Control |
| nav | nav Layer |
| sym | sym Layer |
| a/M | on tap: 'a' and  on hold: 'M' |
